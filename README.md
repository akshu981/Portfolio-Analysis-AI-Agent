This project automates the analysis of institutional investment changes by extracting and comparing 13F regulatory filings from the SEC API. The agent identifies new positions, closed investments, and portfolio modifications between quarters, then maps each investment to its corresponding sector using SIC codes. By reducing manual analysis time from 45 minutes to under 2 minutes (for the given company in code), the system provides instant insights into portfolio turnover, sector allocation shifts, and significant investment changes through an interactive query interface.

Built with Google Gemini 2.5 Flash for cost-effective LLM integration, the agent generates succinct summaries that answer ad-hoc questions about portfolio changes, sector trends, and investment magnitudes. The pipeline handles data deduplication, options filtering, and CUSIP-to-sector mapping to ensure accurate analysis across 20+ industry sectors. The next step for this project is productionizing the system into a web application to make it accessible to a broader audience of investors and analysts.
